# 心电学堂 — AI 心电图判读教学平台

> 面向全体临床医生的微信小程序，AI 苏格拉底式对话教你判读心电图。

## 与 EP Mentor 的关系

```
┌──────────────┐     ┌──────────────┐
│   EP Mentor  │     │   心电学堂    │
│   (Next.js)  │     │  (Taro 小程序) │
│  Web + 管理后台 │     │  微信小程序    │
└──────┬───────┘     └──────┬───────┘
       │                    │
       └────────┬───────────┘
                ▼
    ┌─────────────────────┐
    │  共享后端 (Supabase) │
    │  共享 API (22 路由)  │
    │  共享用户体系         │
    └─────────────────────┘
```

## 技术栈

| 层 | 技术 |
|------|------|
| 前端框架 | Taro 3.x (React) |
| 样式 | Tailwind CSS (weapp-tailwindcss) |
| 后端 | 复用 EP Mentor API (Next.js) |
| 数据库 | 共享 Supabase PostgreSQL |
| AI | 共享 DeepSeek API |
| 登录 | 微信一键登录 |

## 内容体系

| 层级 | 内容 | 定价 |
|------|------|:---:|
| Tier 1 | 基础心电图 (120 例) | 免费 |
| Tier 2 | 心律失常 (80 例) | ¥99/年 |
| Tier 3 | EP 电生理 | 含会员 |
